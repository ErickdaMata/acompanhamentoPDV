JavaScript desabilitado

 

Preciso habilitar scripts?

     Atualmente quase todas as páginas da internet contém JavaScript, uma linguagem de programação que é executada em seu navegador. Esta aplicação depende da habilitação de JavaScript para seu funcionamento, e se estiver desabilitada por alguma razão, sua aplicação não irá funcionar

Aqui você encontra instruções sobre como habilitar (ativar) o JavaScript em seu navegador:

https://www.enable-javascript.com/pt/
